# 重負荷系統の電圧安定性解析ツール

電力系統の系統分離時における電圧安定性をG-Bc平面で解析するインタラクティブなWebアプリケーションです。

## 概要

重負荷・容量性系統において系統分離が発生した際の電圧安定性問題を、G-Bc平面上での可視化とインタラクティブな操作により理解することができるツールです。同期発電機の自己励磁現象や電圧不安定の発生メカニズムを直感的に学習できます。

## 使用方法

### オンライン（推奨）
1. **GitHub Pages**: https://lutelute.github.io/power-system-stability-analysis/ にアクセス
2. すぐにインタラクティブな解析ツールを使用開始

### ローカルでの実行
1. このリポジトリをクローン
```bash
git clone https://github.com/lutelute/power-system-stability-analysis.git
cd power-system-stability-analysis
```
2. `index.html`をブラウザで開く

### 操作方法

#### G-Bc平面での解析
1. **G（コンダクタンス）操作**: 
   - 負荷チェックボックスで各負荷の接続/切離を操作
   - スライダーで直接調整も可能

2. **Bc（容量性サセプタンス）操作**: 
   - コンデンサバンクの投入/切離により容量性負荷を調整
   - ケーブル充電容量の考慮

3. **安定性確認**: 
   - G-Bc平面上の運転点（OP）位置と安定境界円の関係を確認
   - 自己励磁現象の発生条件を視覚的に理解

4. **時間応答シミュレーション**: 
   - シミュレーションボタンで系統分離後の電圧変動を確認

## 特徴

### インタラクティブ解析機能
- **リアルタイムG-Bc平面表示**: パラメータ変更に応じた即座の可視化
- **安定境界円**: 自己励磁現象の発生条件を明確に表示
- **k等高線**: 電圧係数の分布を色分け表示
- **運転点追跡**: 負荷・コンデンサ操作による運転点の移動
- **詳細系統図**: 系統縮約モデルの視覚的表示
- **個別制御**: 各負荷・コンデンサの個別接続/切離操作
- **クイック操作**: ワンクリック安定化・不安定化ボタン
- **固有値解析**: リアルタイム安定性指標計算

### 物理現象の可視化
- **系統縮約モデル**: 発電機-送電線-負荷の簡略化表示
- **電圧係数k**: 負荷端電圧と発電機内部電圧の比の計算
- **固有値表示**: 安定性判定のための数学的指標
- **時間応答**: 系統分離後の動的挙動シミュレーション

## 技術的詳細

### 理論背景
- **G (コンダクタンス)**: G = P/V² で計算される有効電力負荷のアドミタンス表現
- **Bc (容量性サセプタンス)**: Bc = Qc/V² で計算される進み無効電力のアドミタンス表現
- **安定境界**: 同期発電機の自己励磁現象発生条件を示す円
- **電圧係数k**: k = |VL|/|V| = 負荷端電圧/発電機内部電圧の比（k>1で過電圧）

### 解析対象
- **重負荷系統**: 系統分離により周波数維持のための負荷遮断が必要な状況
- **容量性系統**: 電力用コンデンサやケーブル充電容量による進み無効電力
- **系統分離**: 広域系統からの解列による孤立運転状態
- **自己励磁現象**: 過励磁による電圧不安定・発散現象

### 実装技術
- **React 18+**: Hooks使用によるモダンな状態管理  
- **TailwindCSS**: ユーティリティファーストのスタイリング
- **SVG可視化**: G-Bc平面の高精度2D描画
- **数値シミュレーション**: 微分方程式の数値解法とリアルタイム計算
- **レスポンシブデザイン**: PC・タブレット対応UI
- **CDN配信**: 外部依存なしの純粋HTML/CSS/JSアプリケーション
